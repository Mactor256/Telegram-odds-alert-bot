# Telegram Odds Bot ⚽


Telegram bot that finds betting opportunities with odds 2.00+.


## Features

✅ Telegram commands

✅ Odds filtering

✅ PostgreSQL database

✅ Docker support

✅ GitHub Actions


## Commands


/start

Show bot status


/odds

Show available picks



## Installation


Clone repository:


git clone https://github.com/Mactor256/Telegram-odds-alert-bot



Create environment:


cp .env.example .env



Edit:

BOT_TOKEN

ODDS_API_KEY



Run:


docker compose up --build



## Deployment


Push to GitHub.

Use VPS + Docker for 24/7 operation.
